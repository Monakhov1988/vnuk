# Маркетинговый план продвижения «Внучок»

## 1. Целевая аудитория

### Сегмент A: Эмоциональный покупатель
- **Возраст:** 35-50 лет
- **Пол:** 70% женщины
- **Доход:** средний+ (60-150K руб/мес)
- **География:** Москва, СПб, города 500K+
- **Боли:** нехватка времени на родителей, чувство вины, тревога за здоровье
- **Триггеры:** праздники (ДР, 8 марта, День матери/отца), сезонные обострения
- **Поисковое поведение:** «что подарить маме», «подарок папе», «забота о родителях»

### Сегмент B: Tech-savvy покупатель
- **Возраст:** 30-45 лет
- **Пол:** 60% мужчины
- **Доход:** выше среднего (100-250K руб/мес)
- **Боли:** неэффективность ручного контроля, нужен автоматизированный инструмент
- **Триггеры:** знакомство с AI-инструментами, статьи про нейросети
- **Поисковое поведение:** «AI помощник», «нейросеть для семьи», «GPT ассистент»

## 7. A/B тестирование лендингов

### Настройка
- `/` — рандомное распределение 50/50 (сохраняется в localStorage)
- `/a` — принудительно эмоциональный вариант (для Директа, посевов в family-каналах)
- `/b` — принудительно технологический вариант (для tech-каналов, AI-аудитории)

### Что отслеживаем
1. **landing_view** — просмотр лендинга (вариант A/B)
2. **cta_click** — клик по CTA (какая кнопка)
3. **scroll_depth** — глубина прокрутки
4. **auth_start** — начало регистрации
5. **trial_start** — активация trial
6. **paid_convert** — конверсия в оплату

### Критерии принятия решения
- Минимальная выборка: 500 уникальных посетителей на каждый вариант
- Статистическая значимость: p < 0.05
- Основная метрика: CVR в trial (landing_view → trial_start)
- Вторичная метрика: CVR в paid (trial_start → paid_convert)

### UTM-разметка
- Директ A: `?utm_source=yandex&utm_medium=cpc&utm_campaign=gift_care&utm_content=landing_a`
- Директ B: `?utm_source=yandex&utm_medium=cpc&utm_campaign=ai_agent&utm_content=landing_b`
- Telegram: `?utm_source=telegram&utm_medium=seed&utm_campaign=channel_name&utm_content=landing_a`
- Инфлюенс: `?utm_source=influencer&utm_medium=integration&utm_campaign=blogger_name&utm_content=landing_b`
